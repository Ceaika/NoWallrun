# NoWallrun

A client-side Titanfall 2 / Northstar mod that disables wallrunning.

Ever wanted to play Titanfall like a grounded shooter?  
Me neither.

This mod removes wallrunning for the local player. Movement is restricted to ground-based mechanics, changing how fights play out and forcing a slower, more deliberate playstyle.

## Features

- Disables wallrunning
- Client-side only
- Does not affect other players

## Installation

Place the mod folder into:

R2Northstar/mods/

## Status

Work in progress. Behavior may vary depending on scripting limitations and client/server interaction.
